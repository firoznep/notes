Desktop GPO

Create a File Share

  • Create a New Folder under Pictures folder (or any drive) inside your server
  • Share the folder by right-clicking it and selecting Properties.
  • Under the Desktop Background Properties Window select the Sharing Tab and click on the Advanced Sharing button.
  • Tick mark the Share this folder checkbox and then click on the Permissions button.
  • By default, Everyone has Read permissions. To be a little bit more secure lets delete the group by highlighting Everyone group and clicking on the Remove button.
  • Now lets click on Add to add the "Authenticated Users".
  • Make sure "Authenticated Users" have Read Permissions and click on Apply and OK buttons.

Configure the Group Policy Object

  • Open Server Manager > Tools > Group Policy Management.
  • Right-click on "OU" that you want to apply the GPO, and choose to Create a GPO in this domain, and Link it here...
  • Under "new GPO" dialog. Give the name "Desktop Background" and click OK.
  • Right-click on the newly created "Desktop Background" policy and choose "Edit".
  • Under User Configuration navigate to Policies > Administrative Templates > Desktop > Desktop. Double click on Desktop Wallpaper to edit the setting.
  • Check "enabled" and give the Wallpaper full path.("\\hostname\username\pictures\mypic.jpg")
  • Open a command prompt and type "gpupdate /force" and "gpresult /r" to see the group policy has been applied.
  • in user pc type "gpupdate" and restart.
